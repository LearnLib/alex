Project setup

Project setup

To create a new project enter a name of your choice (here ToDo) and the URL where the ToDo is reachable (http://localhost:9090). Optionally, enter a description and press the Create Project button. A message pops up and the project should is listed on the right side.

Click on the name of the newly created project in the list to open it. You are being redirected to the projects dashboard where a brief overview of the project and current learn processes are displayed. On the left, new navigation items are shown to navigate through ALEX.
